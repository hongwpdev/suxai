### SCR-001-A. 수질 탭

**탭 활성 조건**: 기본 탭 (첫 진입 시 자동 선택)

```
┌─────────────────────────────────────────┐
│  실시간 수질 현황          [조회]         │
│  최신 데이터 순                           │
├──────────────────────────────────────── ┤
│ ▌ #1  파주1취수장                        │  ← 위험 카드 (빨강 보더)
│     경기 파주시 | 2026-05-23 22시        │
│         pH       탁도      잔류염소       │
│        7.20    0.0023 NTU  0.1500 mg/L  │
│                              [정수]      │
├─────────────────────────────────────────┤
│   #2  반월정수장                          │  ← 정상 카드 (보더 없음)
│     경기 안산시 | 2026-05-23 22시        │
│         pH       탁도      잔류염소       │
│        7.10    0.0010 NTU  0.5000 mg/L  │
│                              [원수]      │
└─────────────────────────────────────────┘
```

**수질 카드 구성 요소**

| 요소 | 내용 |
|------|------|
| 순번 | `#N` (모노스페이스, 보조색) |
| 시설명 | 14px 굵음 |
| 위치·시간 | `주소 \| 측정시간` (11px, 보조색) |
| pH | 소수점 2자리 표시 |
| 탁도 | 소수점 4자리 + `NTU` |
| 잔류염소 | 소수점 4자리 + `mg/L` |
| 구분 배지 | `liIndDivName` (정수/원수 등) |

**카드 상태별 스타일**

| 상태 | 조건 | 좌측 보더 |
|------|------|---------|
| 위험 | crit 항목 1개 이상 | 3px 빨강 `#e8453c` |
| 주의 | warn 항목 1개 이상 (crit 없음) | 3px 주황 `#f5a623` |
| 정상 | 모든 항목 정상 | 없음 (투명) |

**로딩 상태**: 스켈레톤 카드 5개 표시 (shimmer 애니메이션)

### SCR-001-B. 유량 탭

```
┌─────────────────────────────────────────┐
│  실시간 유량 현황          [조회]         │
│  최신 데이터 순                           │
├─────────────────────────────────────────┤
│   #1  파주정수장                          │
│     한강수계 파주 유입유량                 │
│         측정값         측정시간           │
│      1250.50 ㎥/분   2026-05-23 22시    │
│                              [순간]      │
└─────────────────────────────────────────┘
```

**유량 카드 구성 요소**

| 요소 | 내용 |
|------|------|
| 순번 | `#N` |
| 시설명 | 14px 굵음 |
| 항목명 | `dataItemDesc` (측정 항목 설명) |
| 측정값 | `값 단위` |
| 측정시간 | `occrrncDt` 포맷 |
| 유형 배지 | `순간` (초록) / `적산` (주황) |

### SCR-001-C. AI 분석 탭

**초기 상태**: 빈 화면 + "분석 실행" 버튼

**분석 실행 중**: 스켈레톤 패널 (7줄 shimmer)

```
┌─────────────────────────────────────────┐
│  AI 이상징후 분석          [분석 실행]    │
│  수질·유량 데이터를 AI가 종합 분석합니다 (5분 캐시) │
├─────────────────────────────────────────┤
│  [Groq AI]  수질 8건 · 유량 12건 · 2026-05-23 22:30 기준 │
│                                         │
│  **1. 전체 현황**                        │
│  현재 파주정수장의 수질 상태는 전반적으로  │
│  양호한 수준입니다. pH 및 잔류염소 수치는  │
│  정상 범위 내에 있습니다.                 │
│                                         │
│  **2. 이상징후**                         │
│  이상징후 없음                            │
│                                         │
│  **3. 권고사항**                         │
│  특이사항 없음                            │
└─────────────────────────────────────────┘
```

**AI 분석 패널 구성 요소**

| 요소 | 내용 |
|------|------|
| AI 배지 | `Groq AI` (보라 배지) |
| 데이터 건수 | `수질 N건 · 유량 N건` |
| 기준 시각 | `YYYY-MM-DD HH:mm 기준` |
| 분석 본문 | `**볼드**` → `<b>` 변환 렌더링, 줄바꿈 유지 |

## 4. 상태별 UI 처리

| 상황 | 처리 방식 |
|------|----------|
| 데이터 로딩 중 | 상단 프로그레스바 + 스켈레톤 카드 |
| 데이터 없음 | `"데이터가 존재하지 않습니다."` 중앙 표시 |
| API 호출 완료 | 프로그레스바 100% → 페이드아웃 |
| AI 분석 중 | 스켈레톤 패널 7줄 |

## 5. 인터랙션 흐름

```
[접속]
  ↓
정수장 목록 자동 로드
  ↓
첫 번째 정수장 자동 선택
  ↓
수질·유량 동시 조회
  ↓
[수질 탭] 기본 표시
  │
  ├─ 드롭다운 변경 → 수질·유량 재조회
  │
  ├─ [유량] 탭 클릭 → 유량 카드 표시
  │
  ├─ [AI 분석] 탭 클릭 → 빈 패널
  │     └─ [분석 실행] 클릭 → AI 분석 요청 → 리포트 표시
  │
  └─ [조회] 버튼 클릭 → 현재 탭 데이터 재조회
```
