### SCR-001-공통. 날짜 선택바

```
┌────────────────────────────────────────────────────┐
│  조회 기간  [2026-05-18] ~ [2026-05-25]            │
│            [오늘]  [7일 ▶active]  [30일]  [90일]   │
└────────────────────────────────────────────────────┘
```

| 요소 | 타입 | 동작 |
|------|------|------|
| 레이블 | 텍스트 | `조회 기간` |
| 시작일 입력 | `<input type="date" id="dateFrom">` | 날짜 직접 선택 |
| 구분자 | 텍스트 | `~` |
| 종료일 입력 | `<input type="date" id="dateTo">` | 날짜 직접 선택 |
| 퀵 버튼 | `<button>` × 4 | 오늘 / 7일 / 30일 / 90일 |

**퀵 버튼 동작**

| 버튼 | dateFrom | dateTo | 기본 활성 |
|------|----------|--------|----------|
| 오늘 | 오늘 날짜 | 오늘 날짜 | — |
| 7일 | 오늘 - 6일 | 오늘 | ✔ (초기값) |
| 30일 | 오늘 - 29일 | 오늘 | — |
| 90일 | 오늘 - 89일 | 오늘 | — |

- 날짜를 직접 입력하면 퀵 버튼 active 상태 해제
- 날짜 미입력 상태로 조회 시 백엔드 기본값(현재 시각 기준 최근 1시간) 자동 적용

### SCR-001-A. 수질 탭

**탭 활성 조건**: 기본 탭 (첫 진입 시 자동 선택)

```
┌─────────────────────────────────────────┐
│  실시간 수질 현황          [조회]         │
│  최신 데이터 순                           │
├──────────────────────────────────────── ┤
│ ▌ #1  파주1취수장                        │  ← 위험 카드 (빨강 보더)
│     경기 파주시 | 2026-05-23 22시        │
│         pH       탁도      잔류염소       │
│        7.20    0.0023 NTU  0.1500 mg/L  │
│                              [정수]      │
├─────────────────────────────────────────┤
│   #2  반월정수장                          │  ← 정상 카드 (보더 없음)
│     경기 안산시 | 2026-05-23 22시        │
│         pH       탁도      잔류염소       │
│        7.10    0.0010 NTU  0.5000 mg/L  │
│                              [원수]      │
└─────────────────────────────────────────┘
```

**수질 카드 구성 요소**

| 요소 | 내용 |
|------|------|
| 순번 | `#N` (모노스페이스, 보조색) |
| 시설명 | 14px 굵음 |
| 위치·시간 | `주소 \| 측정시간` (11px, 보조색) |
| pH | 소수점 2자리 표시 |
| 탁도 | 소수점 4자리 + `NTU` |
| 잔류염소 | 소수점 4자리 + `mg/L` |
| 구분 배지 | `liIndDivName` (정수/원수 등) |

**카드 상태별 스타일**

| 상태 | 조건 | 좌측 보더 |
|------|------|---------|
| 위험 | crit 항목 1개 이상 | 3px 빨강 `#e8453c` |
| 주의 | warn 항목 1개 이상 (crit 없음) | 3px 주황 `#f5a623` |
| 정상 | 모든 항목 정상 | 없음 (투명) |

**수질 항목별 판정 기준 (프론트엔드 색상 결정 로직)**

> 카드 및 수치 색상은 아래 임계값 기준으로 JS에서 직접 계산한다.

| 항목 | 위험 (`c-crit`) | 주의 (`c-warn`) | 정상 (`c-ok`) |
|------|--------------|--------------|-------------|
| pH | < **5.8** 또는 > 9.0 | < 6.5 또는 > 8.5 | 6.5 ~ 8.5 |
| 탁도 | > **1.0** NTU | > 0.5 NTU | ≤ 0.5 NTU |
| 잔류염소 | < **0.05** 또는 > 4.0 mg/L | < 0.1 또는 > 2.0 mg/L | 0.1 ~ 2.0 mg/L |

- 카드 클래스: 3항목 중 하나라도 `c-crit` → 카드 `crit` / `c-warn` 존재 → 카드 `warn` / 모두 `c-ok` → 카드 `ok`
- AI 분석 프롬프트의 수질 기준(`pH 6.5~8.5 / 탁도 0.5 NTU / 잔류염소 0.1~2.0`)과 다름에 유의 (AI 프롬프트는 WHO 기준, 카드 판정은 운영 기준)

**로딩 상태**: 스켈레톤 카드 5개 표시 (shimmer 애니메이션)

**초기 상태**: 빈 화면 + "분석 실행" 버튼

**분석 실행 중**: 스켈레톤 패널 (7줄 shimmer)

```
┌─────────────────────────────────────────┐

│  [Groq AI]  수질 8건 · 유량 12건 · 2026-05-23 22:30 기준 │
│                                         │
│  **1. 전체 현황**                        │
│  현재 파주정수장의 수질 상태는 전반적으로  │
│  양호한 수준입니다.                       │
│                                         │
│  **2. 이상징후**                         │
│  이상징후 없음                            │
│                                         │
│  **3. 권고사항**                         │
│  특이사항 없음                            │
└─────────────────────────────────────────┘
```

**AI 분석 패널 구성 요소**

| 요소 | 내용 |
|------|------|
| AI 배지 | `Groq AI` (보라 배지) |
| 데이터 건수 | `수질 N건 · 유량 N건` |
| 기준 시각 | `YYYY-MM-DD HH:mm 기준` |
| 분석 본문 | `**볼드**` → `<b>` 변환 렌더링, 줄바꿈 유지 |

## 5. 인터랙션 흐름

```
[접속]
  ↓
정수장 목록 자동 로드
  ↓
첫 번째 정수장 자동 선택
  ↓
날짜 기본값 자동 설정 (오늘 기준 최근 7일)
  ↓
수질·유량 동시 조회 (startDate, endDate 포함)
  ↓
[수질 탭] 기본 표시
  │
  ├─ 드롭다운 변경 → 수질·유량 재조회 (현재 날짜 기간 유지)
  │
  ├─ 날짜 퀵 버튼 클릭 → 날짜 범위 변경
  │     └─ [조회] 버튼 클릭 → 새 기간으로 API 재호출
  │
  ├─ 날짜 직접 입력 → 퀵 버튼 active 해제
  │     └─ [조회] 버튼 클릭 → 새 기간으로 API 재호출
  │
  ├─ [유량] 탭 클릭 → 유량 카드 표시
  │
  ├─ [AI 분석] 탭 클릭 → 빈 패널
  │     └─ [분석 실행] 클릭 → AI 분석 요청 (현재 날짜 기간 포함) → 리포트 표시
  │
  └─ [조회] 버튼 클릭 → 현재 탭·현재 날짜 기간으로 데이터 재조회
```
